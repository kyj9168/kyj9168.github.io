git 블로그 관리시 hexo 프레임워크를 사용한다면
hexo-admin을 사용해보면 편하다.



```bash
npm install --save hexo-admin
```

위 명령어로 git 블로그 프로젝트에 설치하고 아래 url로 진입하면된다.

http://localhost:4000/admin

---
새로운 포스트를 바로 웹에서 작성 할 수 있다.
